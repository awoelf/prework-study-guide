HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The id attribute is a way of identifying and selecting elements.

CSS

  • A margin indicated how much space we want around the outside of an element.
  • A padding indicated how much space we want around the content inside an element.
  • You can link an external CSS file in the head section of the HTML file.
  • Create style rules for class elements by using a . before the name of the class.

Git

  • git status: check what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commit all files in the working branch.
  • git commit -m "commit message": commit changes
  • git pull origin main: make sure current branch is up to date
  • git push origin feature/add-html: push changes made in branch
  • git checkout main, git pull origin main: checkout and update local files with main branch

JavaScript

  • The script element is placed within the body element and below the footer.
